Overview
========
Galapago is an HTML5 game for interactive television. Time Savers (TS) is developing Galapago for TransGaming (TG), a maker of interactive games. Galapago already exists as a PC game. This Galapago is mostly just a rewrite of the existing PC game, with some additions such as such as support for Google ads. The game is required to be built using the HTML5 hardware-accelerated <canvas> tag. New generations of interactive televisions such Sony Blu-Ray (Opera), Dish Network Hopper (Webkit?), and Toshiba (Espial Webkit) support HTML5 specs in general, and in particular <canvas>.
